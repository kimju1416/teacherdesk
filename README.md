# 🖥️ TeacherDesk — 선생님 바탕화면 대시보드

시간표 · 학사일정 · 급식 · 날씨 · 할 일 · 명렬표 · 내선번호를 바탕화면에 항상 띄워주는 **선생님 전용 데스크톱 프로그램**입니다.

Lively Wallpaper 같은 월페이퍼 프로그램 없이, 설치 한 번으로 끝나요.

## 📥 설치 방법

1. [**Releases 페이지**](../../releases/latest)에서 `TeacherDesk-Setup-x.x.x.exe` 다운로드
2. 더블클릭 → 자동 설치 & 실행
   - "Windows에서 PC를 보호했습니다" 창이 뜨면 **[추가 정보] → [실행]** 클릭 (개인 제작 프로그램이라 뜨는 안내입니다)
3. 설정 마법사에서 학교 검색 → 시간표·학사일정 업로드 → 테마 선택 → **🚀 대시보드 생성하기**
4. 끝! 이후엔 컴퓨터를 켜면 자동으로 바탕화면에 대시보드가 깔립니다

## ✨ 주요 기능

- ⏰ 실시간 시계 · 오늘 날짜 · 현재 교시 표시
- 📋 주간 시간표 (AI가 파일에서 자동 인식)
- 🍱 오늘 급식 (아침/점심/저녁 선택 표시, NEIS 자동 연동)
- 📅 학사일정 · D-day
- ☀️ 실시간 날씨
- ✅ 할 일 · 메모 (영구 저장)
- 👥 명렬표 · 📞 내선번호 검색
- 🎨 테마 12종 · 레이아웃 3종 · 🧩 바탕화면에서 카드 드래그·크기 조절
- 📅 달력 위젯 (학사일정 + 개인 일정 추가/수정)
- 🔄 **자동 업데이트** — 새 버전이 나오면 알아서 받아서 다음 실행 때 적용

## 🖱️ 사용 팁

| 하고 싶은 것 | 방법 |
|---|---|
| 대시보드를 잠깐 맨 앞으로 | `Ctrl + Alt + D` (다시 누르면 원위치) |
| 새로고침 / 다시 설정 / 종료 | 트레이(작업표시줄 오른쪽 끝) 보라색 아이콘 우클릭 |
| 다른 컴퓨터로 옮기기 | 트레이 → 💾 설정 내보내기 → 새 컴퓨터에서 📂 설정 가져오기 |
| 표시할 모니터 바꾸기 | 트레이 → 🖥️ 표시할 모니터 |

## 🔑 필요한 API 키 (무료)

| 키 | 발급처 | 용도 |
|---|---|---|
| NEIS API 키 | [open.neis.go.kr](https://open.neis.go.kr) | 급식 조회 |
| Gemini API 키 | [aistudio.google.com](https://aistudio.google.com) | 시간표·일정 파일 AI 인식 |
